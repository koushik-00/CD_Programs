# CD_Programs
**Execution of Lex Programs**
```bash
  lex filename.l
```
```bash
  gcc lex.yy.c
```
```bash
  ./a.out
```
**Execution of Yacc Programs**
```bash
  lex filename.l
```
```bash
  yacc -d filename.y
```
```bash
  gcc lex.yy.c y.tab.c -w
```
```bash
  ./a.out
```

**Unix Path system to reach Desktop in windows**
```bash
  /mnt/c/users/Desktop
 ```
